EJERCICIO 2

Un electrodoméstico tiene una potencia de 1500 vatios y se usa durante 3 horas al día. Diseña un 
programa que calcule el consumo de energía en kilovatios-hora (kWh) al mes, suponiendo que el mes 
tiene 30 días.

1.Iniciar
2.Convertir la potencia de vatios a kilovatios.
3.Calcular el consumo diario en kilovatios-hora (kWh).
4.Calcular el consumo mensual multiplicando el consumo diario por los días del mes.
5.Mostrar el consumo mensual en kWh.
6.Finalizar

Proponer Pseudocodigo
program calcularConsumoDeEnergiaEnKwhAlMes
star

// Programa para calcular el consumo de energía en kWh al mes
// Declaración de variables
Declare potenciaEnW, horasPorDia, diasPorMes, consumoMensual

// Asignar valores iniciales
potenciaEnW = 1500
horasPorDia = 3
diasPorMes = 30

// Convertir vatios a kilovatios
potenciaEnkW = potenciaEnW / 1000

// Calcular el consumo mensual en kWh
consumoMensual = potenciaEnkW * horasPorDia * diasPorMes

// Mostrar el resultado
Display "El consumo mensual de energía es: ", consumoMensual, " kWh"

End
